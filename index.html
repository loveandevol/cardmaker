SNS 카드 만들기

사용 방법

  1. 스킨 세팅(1회): 맨 아래 설정 카드에서 CSS/JS를 복사해 각자 스킨의 html의 </body> 바로 위와 css의 가장 아래에 넣습니다.
  2. 사진 첨부: 외부 URL을 넣거나 폴더 아이콘을 눌러 PC에서 업로드하세요.
  3. 본문 복사: 완성 후 [본문 복사]를 눌러 티스토리의 글쓰기→html모드 또는 html 블럭에 붙여넣습니다.

주의사항

  • html 블럭으로 넣은 코드는 댓글 더보기가 작동하지 않을 수 있어요.
  • 게시물 내용이나 댓글에 {{텍스트}} 형식으로 입력하면 파란색으로 강조됩니다.
  • 이미지 저장 시 이미지 첨부에 외부 URL을 사용한 경우 브라우저 보안 정책(CORS)으로 인해 저장이 안 될 수 있습니다. (이 경우 PC 업로드 권장)
